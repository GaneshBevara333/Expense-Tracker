# Expense Tracker

A simple Expense Tracker built in C++ using file handling.

## Features

- Add expenses
- View all expenses
- Search by category
- Calculate total spending
- Delete expenses
- Persistent storage using text files

## Technologies

- C++
- File Handling
- Structures
- STL Vector

## How to Run

```bash
g++ expense_tracker.cpp -o expense_tracker
./expense_tracker
